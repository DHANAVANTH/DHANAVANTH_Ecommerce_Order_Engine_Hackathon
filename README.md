1. Project Overview

This project is an E-commerce Order Engine designed to manage customer orders efficiently. It handles order creation, processing, and tracking in a structured and scalable way.

The system simulates real-world e-commerce workflows such as order placement, validation, and status updates.

2. Features Implemented
 Order creation and management
 Order status tracking (Placed, Shipped, Delivered)
 Customer details handling
 price calculation logic
 Basic validation and error handling
 Modular and structured code design

3.Design Approach
Followed Object-Oriented Programming (OOP) principles
Used layered structure:
Controller (handling requests)
Service (business logic)
Model (data representation)
Code is written to be scalable and maintainable
Focused on clean structure and separation of concerns

4.Assumptions
No real database is used (data may be stored in memory or simple structures)
Single user system (no authentication implemented)
Payment integration is not included
Focus is mainly on backend logic
